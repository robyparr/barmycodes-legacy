# barmycodes
A simple barcode generator service.


## Getting started

Setup `virtualenv` if not already done:

```
virtualenv env
```

Activate `virtualenv`:

```
source ./env/bin/activate
```

Install dependencies:

```
pip install -r requirements.txt
pip install -e .
```

Run the tests:

```
python setup.py test
```

Run the app:

```
./run-app.sh
```
